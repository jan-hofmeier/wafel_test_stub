# wafel_test_stub

This plugin makes the IOSU-TEST process exit immediately and handles the messages with a MCP thread 

## How to use

- Copy the `wafel_test_stub.ipx` to `/wiiu/ios_plugins`

## Building

```bash
export STROOPWAFEL_ROOT=/path/too/stroopwafel-repo
make
```
